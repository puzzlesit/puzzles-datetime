# puzzles-datetime

## Installation
```
npm install --save puzzles-datetime
```

### Import the package in your main.js file
```
import PuzzlesDatetime from 'puzzles-datetime'
```

### To install it globally
```
Vue.use(PuzzlesDatetime)
```

### Or in locally in a component
```
import 'PuzzlesDatetime' from 'puzzles-datetime'
```

### Example
```
<puzzles-datetime />
```
